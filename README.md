# Smart Hostel Room Allocation System

This is a simple web-based project to manage hostel rooms and allocate rooms to students based on their requirements.

The main purpose of this project is to show room allocation logic in a clear and easy way.

---

## Features

- Add new hostel rooms
- View all available rooms
- Allocate room based on:
  - Number of students
  - AC requirement
  - Attached washroom requirement
- Automatically allocates the smallest suitable room
- Shows message if no room is available

---

## Technology Used

- HTML
- CSS
- JavaScript

(No framework is used to keep the logic simple and clear)

---

## How It Works

1. User adds rooms with capacity and facilities.
2. All rooms are stored in a list.
3. While allocating a room:
   - Conditions are checked (capacity, AC, washroom)
   - Smallest suitable room is selected
4. If no room matches the requirement, a message is shown.

---

## How to Run the Project

1. Download or clone the repository
2. Open `index.html` in any web browser
3. Add rooms and allocate rooms using the UI
